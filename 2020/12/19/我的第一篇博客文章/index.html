## 创建 hexo 博客教程

1、安装 node.js

2、npm install -g cnpm –registry=http://registry.npm.taobao.org 安装淘宝 cnpm 管理器

3、cnpm install -g hexo-cli 安装 hexo 框架

4、mkdir F:blog 创建 blog 目录

5、cd 到 blog 目录

6、hexo init 初始化博客

7、hexo s 启动本地博客服务

8、localhost:4000 本地访问地址

9、hexo n “填文章名字” 创建新的文章

10、hexo clean 清理

11、hexo g 生成

12、GitHub 创建一个新仓库 （仓库名为 GitHub 昵称+github.io）

13、cnpm install –save hexo-deployer-git 在 blog 目录下安卓 git 部署插件

14、配置 _config.yml

deploy:
type: git
repo: https://github.com/liu-v0916/liu-v0916.github.io.git
branch: master

15、hexo d 部署到 GitHub 仓库（就可以访问”http://liu-v0916.github.io"来查看博客）

16、git clone 主题地址 themes/主题名称 (下载主题到本地)

17、修改_config.yml
theme: yilia

18、hexo clean 清理

19、hexo g 生成

20、hexo d 部署到远端
