百度定位功能

一、申请 API Key

​ 1、注册成为百度开发者

​ 2、获取 SHA1

​ 打开 Android Studio 中的任意一个项目，点击右侧工具栏的 Gradle→ 项目名 →:app→Tasks→ android->signingReport 双击 signingReport

​ 3、申请 API Key

二、下载 SDK，和准备工作

​ 1、把压缩包中的 jar 文件放到 app>libs 目录下

​ 2、在 src>main 目录下新建一个 jniLibs 文件夹，把剩余的 so 文件放入其中

​ 3、在 app>build.gradle 中配置 compile fileTree(dir: ‘libs’, include: [‘*.jar’])
